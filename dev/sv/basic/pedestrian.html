3.1. pgr_dijkstra¶

Dijkstra-algoritmen var den första algoritmen som implementerades i pgRouting. Den kräver inga andra attribut än identifierarna id, källa och mål samt vikterna kostnad och omvända_kostnader.

Du kan ange när grafen ska betraktas som ”riktad” <https://en.wikipedia.org/wiki/Directed_graph>`__ eller ”oriktad”.

Sammanfattning av signatur

pgr_dijkstra(Edges SQL, start_vid,  end_vid  [, directed])
pgr_dijkstra(Edges SQL, start_vid,  end_vids [, directed])
pgr_dijkstra(Edges SQL, start_vids, end_vid  [, directed])
pgr_dijkstra(Edges SQL, start_vids, end_vids [, directed])
pgr_dijkstra(Edges SQL, Combinations SQL [, directed])

RETURNS SET OF (seq, path_seq, start_vid, end_vid, node, edge, cost, agg_cost)
    OR EMPTY SET

Beskrivning av funktionen finns i pgr_dijkstra.

Observera

  • Många pgRouting-funktioner har sql::text som ett av sina argument. Även om detta kan verka förvirrande i början gör det funktionerna mycket flexibla eftersom användaren kan skicka en SELECT-sats som funktionsargument så länge det returnerade resultatet innehåller det antal attribut som krävs och de korrekta attributnamnen.

  • De flesta av pgRoutings implementerade algoritmer kräver inte geometri.

  • PgRouting-funktionerna returnerar inte en geometri, utan endast en ordnad lista över noder eller kanter.

3.1.1. Övning 1: Routning av enstaka fotgängare¶

Problem:

  • Går

    • från ”Auckland University of Technology”

    • till ”Four Points by Sheraton”.

  • Calculate routes with costs in length in meters.

Lösning:

  • The pedestrian wants to go from vertex 10840 to vertex 21570 (lines 5 and 6).

  • The pedestrian’s cost is in terms of length. In this case length (line 3).

3.2. pgr_dijkstraKostnad¶

När huvudsyftet är att beräkna den totala kostnaden, utan att ”inspektera” pgr_dijkstra-resultaten, ger pgr_dijkstraCost ett mer kompakt resultat.

Sammanfattning av signatur

pgr_dijkstraCost(edges_sql, start_vid,  end_vid  [, directed])
pgr_dijkstraCost(edges_sql, start_vid,  end_vids [, directed])
pgr_dijkstraCost(edges_sql, start_vids, end_vid  [, directed])
pgr_dijkstraCost(edges_sql, start_vids, end_vids [, directed])
pgr_dijkstraCost(edges_sql, combinations_sql   [, directed])

RETURNS SET OF (start_vid, end_vid, agg_cost)
    OR EMPTY SET

Beskrivning av parametrarna finns i pgr_dijkstraCost
